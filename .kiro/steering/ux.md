# UX Researcher Agent Steering

Use this guide when you need expert UX research assistance: `#ux`

## When to Use the UX Researcher Agent

- User research and insights
- Usability testing
- User journey mapping
- Persona development
- A/B testing analysis
- User feedback analysis

## Research Methodology

### User Interview Planning
- Define research objectives
- Recruit participants
- Create interview guides
- Conduct sessions
- Analyze findings
- Generate insights

### Usability Testing
- Design test tasks
- Recruit participants
- Observe interactions
- Collect feedback
- Identify issues
- Recommend improvements

### Survey Design
- Formulate questions
- Design response scales
- Pilot test
- Distribute survey
- Analyze results
- Report findings

## Yeko User Segments

### Super Admin (Yeko Core)
- **Goals**: Manage global templates, catalogs, and system
- **Pain Points**: Complex configuration, bulk operations
- **Needs**: Efficiency, control, reporting

### School Admin (Yeko School)
- **Goals**: Manage school operations, staff, students
- **Pain Points**: Data entry, reporting, compliance
- **Needs**: Ease of use, automation, insights

### Teachers (Future)
- **Goals**: Manage classes, grades, communication
- **Pain Points**: Time-consuming tasks, parent communication
- **Needs**: Simplicity, mobile access, notifications

### Parents (Future)
- **Goals**: Track student progress, communicate with teachers
- **Pain Points**: Limited visibility, communication delays
- **Needs**: Transparency, mobile-first, notifications

## Research Techniques

### Contextual Inquiry
- Observe users in their environment
- Understand workflows
- Identify pain points
- Discover workarounds
- Gather insights

### Card Sorting
- Understand information architecture
- Test navigation structure
- Validate categorization
- Improve organization
- Inform design

### Tree Testing
- Validate navigation structure
- Test findability
- Identify issues
- Optimize hierarchy
- Improve UX

### Analytics Analysis
- Track user behavior
- Identify patterns
- Find drop-off points
- Measure engagement
- Inform decisions
